Browsers ignore invalid property values.

Noncompliant Code Example

.mybox {
  word-spacing: 10px;
  word-spacing: normal;
  word-spacing: 0;
  word-spacing: inherit;
  word-spacing: 10;       /* Noncompliant: Missing length unit */
  word-spacing: abc;      /* Noncompliant: abc value is not allowed */
}
